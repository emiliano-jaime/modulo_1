# Uso de Colab

Nos metemos a Google Colab, creamos un "notebook"(cuaderno), apretamos editar y vamos a configuración de notebook, ahí ponemos tipo de entorno de ejecución R (además tenemos la opción de Python 3 y Julia), acelerador de hardware CPU (tenemos algunas opciones gratis y otras premium) y la visión de entorno recomendada (última versión). Tocamos +Texto para agregar texto sobre el tema que vamos a realizar, lo hacemos utilizando lenguaje técnico, y también podemos agregar un código de programación tocando +Código.
